Projects

Text Extraction from Image

Text Extraction from Image is a project that focuses on extracting textual information from images and converting it into editable and searchable text. The goal of this project is to develop an application that can process images containing text, use Optical Character Recognition (OCR) techniques to recognize the text within the image, and then provide the extracted text as output.

IMAGE CLASSIFICATION using CNN web application

IMAGE CLASSIFICATION using CNN web application is a project that involves creating a web-based application capable of classifying images using Convolutional Neural Networks (CNN). The goal of this project is to allow users to upload images through the web interface, and the application will use a pre-trained CNN model to predict and display the most likely class or category that the image belongs to.

Multiple object detection using deep learning

Multiple object detection using deep learning is a computer vision task that involves identifying and localizing multiple objects within an image or a video frame. It utilizes deep learning techniques, such as SSD (Single Shot Multibox Detector), to efficiently detect and classify multiple objects of various classes simultaneously, enabling applications like autonomous vehicles, surveillance, and augmented reality.
